# HouseBank
A simple application which calculates the cost from a housing loan.

## Tech
- Git and Github
- Visual Studio
- Visual Studio Code

### Backend =>
- ASP.NET Core Web API
- .NET 7.0
- FluentValidation
- Swagger UI

### Frontend =>
- Typescript
- React 18.2.0
- MaterialUI
- Axios
- React Router
